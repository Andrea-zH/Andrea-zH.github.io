Hugo 博客使用说明

分支说明

分支用途操作对象
master存储博客源码你写文章、提交代码的地方
gh-pages存储构建产物GitHub Actions 自动部署，不要手动操作

重要：永远只操作 master 分支，不要手动提交到 gh-pages 分支！

快速上手

第一步：写文章

  1. 进入文章目录：D:\blog\andrea\content\posts
  2. 新建 .md 文件（或用命令创建：hugo new posts/文章名.md）
  3. 文件开头添加 Front Matter：
+++
date = '2026-03-20T10:00:00+08:00'
draft = false
title = '文章标题'
+++
  1. 在 +++ 下方用 Markdown 写正文
  2. 图片放在文章同级目录，引用方式：![描述](图片名.jpg)

第二步：本地预览（可选）

cd D:\blog\andrea
hugo server -D
# 浏览器访问 http://localhost:1313
# Ctrl+C 停止服务

第三步：发布到网站

# 1. 进入项目目录
cd D:\blog\andrea

# 2. 查看修改内容（确认无误）
git status

# 3. 添加所有修改
git add .

# 4. 提交修改（引号内写说明）
git commit -m "新增春菜文章"

# 5. 推送到 master 分支（重要！不是 gh-pages）
git push origin master

等待 1-2 分钟，访问 https://Andrea-zH.github.io/ 即可查看。
