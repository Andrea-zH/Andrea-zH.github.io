Hugo 博客使用说明（自动部署版）

快速上手

写文章

  1. 在 D:\blog\andrea\content\posts 下创建 .md 文件
  2. 文件开头添加 Front Matter：
    +++
date = '2026-03-20T10:00:00+08:00'
draft = false
title = '文章标题'
+++
  3. 在 +++ 下方用 Markdown 写正文

本地预览（可选）

cd D:\blog\andrea
hugo server -D
# 访问 http://localhost:1313
# Ctrl+C 停止

发布到网站

cd D:\blog\andrea
git add .
git commit -m "新增文章"
git push origin main

等待 1-2 分钟，访问 https://Andrea-zH.github.io/ 即可查看。

工作原理

推送代码到 main 分支后，GitHub Actions 会自动：

  1. 安装 Hugo
  2. 构建静态网页
  3. 部署到 gh-pages 分支

无需手动执行 hugo 命令或操作 public 目录。

注意事项

  1. 草稿文章：draft = true 的文章不会发布，需改为 draft = false
  2. 首次使用：确保已执行 git push origin main（不是 gh-pages）
  3. 部署失败：查看 GitHub 仓库的 Actions 标签页，检查错误日志
  4. 网站 404：检查 Settings → Pages → Source 是否选择 gh-pages 分支
